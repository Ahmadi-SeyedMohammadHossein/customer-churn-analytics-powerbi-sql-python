# 📊 Customer Churn Analytics — End-to-End Data Project

> **Tools:** Python · SQL · Power BI · DAX  
> **Domain:** Telecom / CRM  
> **Type:** End-to-End Personal Project

---

## 🎯 Business Problem

Customer churn is one of the most critical challenges in subscription-based businesses.
This project analyzes a real telecom dataset to answer:

- Why do customers leave?
- Which segments are at highest risk?
- What actions can reduce churn?

---

## 📌 Key Results

| Metric | Value |
|--------|-------|
| Total Customers | 7,043 |
| Churned Customers | 1,869 |
| **Churn Rate** | **26.5%** |
| Highest Risk Segment | Month-to-Month contracts |
| Early Tenure Risk | Customers < 12 months |

---

## 🔍 Key Insights

- 📌 **Month-to-month contracts** have significantly higher churn than annual contracts
- 📌 **Higher monthly charges** (above avg.) correlate strongly with churn probability  
- 📌 **New customers (< 12 months tenure)** are the most vulnerable segment
- 📌 Customers without **tech support or online security** churn at higher rates
